# Så skapar du Core Files i OpenClaw UI

Dashboarden visar fortfarande MISSING – då skapar du filerna manuellt.

**Gör så här för varje fil:**

1. I OpenClaw Control → Agents → main → **Files**, klicka på filnamnet (t.ex. **AGENTS.md**).
2. I rutan till höger (Content) klistrar du in innehållet från **samma fil** i denna mapp (`openclaw/workspace/`).
3. Klicka **Save**.
4. Upprepa för nästa fil.

**Ordning:** Börja med **AGENTS.md** (viktigast), sedan SOUL.md, IDENTITY.md, USER.md, TOOLS.md, HEARTBEAT.md, BOOTSTRAP.md.

Filer att kopiera från (i denna repo):
- `openclaw/workspace/AGENTS.md`
- `openclaw/workspace/SOUL.md`
- `openclaw/workspace/IDENTITY.md`
- `openclaw/workspace/USER.md`
- `openclaw/workspace/TOOLS.md`
- `openclaw/workspace/HEARTBEAT.md`
- `openclaw/workspace/BOOTSTRAP.md`

Öppna varje fil i Cursor, Ctrl+A → Ctrl+C, byt till OpenClaw UI → klistra in i Content → Save.

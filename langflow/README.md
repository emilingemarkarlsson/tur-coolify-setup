# Langflow

Langflow - Visual editor for building LLM applications

## Konfiguration

- **Image:** `langflowai/langflow:latest`
- **Domain:** `langflow.theunnamedroads.com`
- **Status:** Aktiv i Coolify

## Beskrivning

Langflow är ett visuellt verktyg för att bygga och prototypa LLM-applikationer. Det ger en drag-and-drop-gränssnitt för att skapa komplexa AI-arbetsflöden utan kodning.

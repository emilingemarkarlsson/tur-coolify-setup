# N8N Workflow Automation

Workflow automation and integration platform for The Hockey Analytics.

## Deployment
- Domain: `automation.thehockeyanalytics.com`
- Platform: N8N Community Edition
- Timezone: Europe/Stockholm
- Metrics: Enabled

## Features
- Visual workflow builder
- API integrations
- Webhook endpoints
- Scheduled automation
- Data transformation

## Use Cases
- Scraper automation triggers
- Data pipeline orchestration
- API endpoint management
- Notification workflows
- ETL process coordination

## Deploy
Use the provided `docker-compose.yml` in Coolify.
